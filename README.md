# How to create a Maven Archetype



## Directory structure

```
archetype
|  pom.xml
|__src
    |__main
        |__resources
            |__META-INF
            |  |__maven
            |     |__archetype-metadata.xml    
            |__archetype-resource
               |__archetype-metadata.xml
                  |__.gitignore
                  |__README.md
```



#### Archetype Resource

Archetype-resources folder contain your prototype project.
